# OneDrive Storage Root Example

Copy this file to `storage-root.local.md` and update the path for the current Mac.

The real `storage-root.local.md` file is ignored by git.

## Root Path

```text
/path/to/OneDrive/Training Pal
```

## Notes

- This path should point to the local OneDrive-synced `Training Pal` folder.
- Do not commit machine-specific paths, account names, or private folder details unless the trainer explicitly wants them public.
- If Program Pal cannot access this folder because of macOS permissions, ask the trainer to confirm the path or grant access.
