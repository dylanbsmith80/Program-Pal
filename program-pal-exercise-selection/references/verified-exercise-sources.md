# Verified Exercise Sources

Use these sources as trusted references for exercise selection, form checks, substitutions, and coaching notes.

## ACE Fitness Exercise Library

URL: https://www.acefitness.org/resources/everyone/exercise-library/

Use for:

- Exercise options by body part
- Exercise options by equipment
- Difficulty level
- Form descriptions
- Beginner, intermediate, and advanced exercise ideas

## NASM Exercise Library

URL: https://www.nasm.org/resource-center/exercise-library

Use for:

- Step-by-step exercise instructions
- Stretching and exercise technique references
- General exercise form checks

## NSCA Exercise Tutorials And Videos

URL: https://www.nsca.com/education/nsca-videos/

Use for:

- Strength and conditioning exercise tutorials
- Video-based form references
- Coaching and technical context for training movements

## Source Preference Rules

Program Pal should prefer verified sources when choosing or explaining exercises.

If using an exercise from a non-verified source:

- Apply extra caution.
- Check whether ACE, NASM, NSCA, or the trainer's approved notes include a safer or simpler alternative.
- Avoid novelty exercises unless there is a strong client-specific reason.
- Explain why the exercise is appropriate.

Do not copy large exercise libraries into the repository. Use these links as verification sources and build a separate curated local library only when the trainer explicitly approves entries.
